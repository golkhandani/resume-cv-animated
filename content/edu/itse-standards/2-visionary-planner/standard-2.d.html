How this assessment could improve the learning process

1. What is the purpose of your assessment?

  • This assessment will evaluate students' ability to recognize situations that need to be addressed with Array.

2. Identify the content (goals/outcomes) that are going to be assessed by this formative assessment.

  • Recognize gaps between students to support peer learning and reduce performance gaps.
  • Determine effective teaching methods and identify issues in the lesson plan to tailor learning.

3. What factors did you consider when designing your assessment?

  • Focus questions on the concept of Array, not language-specific implementations.
  • Cover various Array use cases and provide hints for clarity.

4. Procedures and responsibilities of both the students and the teacher.

  • Students: Stick to core concepts, use creativity and trial/error techniques.
  • Teachers: Provide hints, remain available, encourage understanding, and boost confidence.

5. How the assessment supports student learning:

  • Allows the teacher to personalize instruction and adjust the lesson plan.
  • Helps students recognize their learning gaps.

Note: Formative assessment helps surface misconceptions and inform instruction. Referencing Dr. Shuchi Grover’s framework.

Array, File and Search Assessment

A) Write a flow of the below application:

A program for a coffee ordering machine that stores the data of each customer throughout the day, along with their orders. Customers' orders should be processed according to the priority they give as input.

  • Design the model and properties for each order.
  • Use files, arrays, and sorting functions to solve the problem.
  • Hints:
    • Consider storage limitations and cleanup operations.
    • Choose one or multiple files strategically to manage flow.

B) At the end of the day, the machine specialist should be able to see the list of customers and their orders. Be creative and optimize the process by reducing redundancy.

C) Which capability of Array should be used for a delete flow?

D) What capability of the Operating System should be used to add a new item to the order list?
